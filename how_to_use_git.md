| 명령어                   | 설명                |
| --------------------- | ----------------- |
| `git init`            | Git 저장소 초기화       |
| `git add .`           | 모든 변경 파일 스테이징     |
| `git commit -m "메시지"` | 변경 사항 커밋          |
| `git push`            | GitHub로 업로드       |
| `git pull`            | GitHub에서 최신 버전 받기 |
| `git status`          | 현재 상태 확인          |
| `git log`             | 커밋 기록 보기          |
